# Contributing to Seedream

Thank you for your interest in contributing! This project welcomes issues, feature requests, and pull requests.

## How to Contribute

### Reporting Bugs

1. Open a [new issue](https://github.com/ElijahZhao/seedream-HuiBenShengCHENG/issues/new) with a clear title.
2. Include your OS, app version, and steps to reproduce.
3. Attach screenshots if possible.

### Suggesting Features

1. Open an issue with the `enhancement` label.
2. Describe the feature and why it would be useful.

### Pull Requests

1. Fork the repository.
2. Create a feature branch (`git checkout -b feat/your-feature`).
3. Commit your changes with clear messages.
4. Open a Pull Request against `main`.

## Development Setup

```bash
git clone https://github.com/ElijahZhao/seedream-HuiBenShengCHENG.git
cd seedream-HuiBenShengCHENG
pnpm install
pnpm tauri:dev   # Desktop dev mode
```

## Code Style

- TypeScript strict mode enabled
- Use ESLint and Prettier for formatting
- Follow the existing patterns in `src/`
